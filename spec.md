# Vandana Veg Restaurant Website

## Overview
A modern, animated website for Vandana Veg Restaurant, a pure vegetarian casual dining restaurant located in Pratap Nagar, Jaipur. The website features premium animations, glassmorphism UI, and a dark gradient theme with leafy green, orange, cream, and berry color palette.

## Core Features
- Fully responsive design for all screen sizes
- Dark gradient background with glassmorphism UI elements
- Smooth animations and micro-interactions throughout
- Modern typography using Poppins and Montserrat fonts
- Parallax scrolling and motion effects

## Website Sections

### Hero Section
- Fullscreen cinematic presentation with animated floating elements
- Headline reveal animations
- Glowing "Reserve a Table" call-to-action button
- Particle effects and depth motion for immersive experience

### About Us
- Image grid with hover zoom effects
- Fade-in text animations highlighting restaurant values
- Focus on freshness, hygiene, and family-friendly atmosphere

### Our Special Menu
- Animated food cards with bounce-in effects
- Interactive hover effects with lift and shadow
- Display of signature vegetarian dishes

### Why Choose Us
- Four glassmorphism cards featuring key benefits:
  - Pure Vegetarian
  - Hygiene Standards
  - Authentic Taste
  - Family-Friendly Environment
- Animated icons with hover interactions

### Food Gallery
- Image gallery with hover zoom effects
- Lightbox functionality with smooth transitions
- Showcase of restaurant dishes and ambiance

### Customer Reviews
- Glass-style review cards
- Animated slider with fade transitions
- Display customer testimonials and ratings

### Location Section
- Embedded Google Map showing restaurant location in Pratap Nagar, Jaipur
- Interactive map with restaurant marker

### Contact Section
- Animated contact form with fields for:
  - Name
  - Email
  - Message
- Submit button with glow and ripple animations
- Restaurant contact information display

### Footer
- Elegant glassmorphism footer design
- Glowing social media icons
- Quick navigation links
- Restaurant details and hours

## Animations & Interactions
- Smooth scroll behavior between sections
- Section animations triggered on scroll
- Hover effects including lifts, glows, and scaling
- Floating background elements (herbs/leaves)
- Page transition animations
- Interactive button states with ripple effects

## Technical Requirements
- Component-based React architecture
- Tailwind CSS for styling
- Framer Motion for animations
- Responsive design implementation
- Performance optimization for smooth animations
